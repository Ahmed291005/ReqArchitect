# **App Name**: ReqBot Chat

## Core Features:

- Chat Interface: Landing page with a chat interface to display messages in speech bubbles with timestamps.
- Dark/Light Mode Toggle: Toggle for switching between light and dark modes using Tailwind CSS.
- AI Chat Backend: AI chat backend powered by the OpenAI API (gpt-4 or gpt-4o) to elicit requirements as a Business Analyst (ReqBot).
- Requirement Output: Use tool-like reasoning to determine when requirements are complete and can be output in a JSON format with specified fields.
- Requirement Classification: After taking requirements, classify them into types of requirements (e.g., functional, non-functional).

## Style Guidelines:

- Dark color scheme to complement the chat interface and enhance readability.
- Primary color: #9DD6FF (light blue), a vibrant but friendly color reminiscent of a helpful chatbot assistant. The light blue works well with both light and dark themes.
- Background color: #13161A (very dark desaturated blue), providing a clean background that ensures good contrast for light text.
- Accent color: #F9C063 (gold), used to highlight interactive elements and calls to action.
- Body and headline font: 'Inter', a sans-serif typeface known for its clean and modern design that ensures excellent readability across various screen sizes.
- Code font: 'Source Code Pro', a monospaced font ideal for displaying code snippets and JSON outputs within the chat interface.
- Simple, clear icons for user actions and system messages to enhance usability.
- Clean and intuitive layout with speech-bubble style messages and clear timestamps for easy readability.
- Subtle animations for message delivery and user interactions to provide a smooth and engaging experience.
